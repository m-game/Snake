# Snake

![Snake](https://raw.githubusercontent.com/muumlover/pics/main/pygame-snake.jpg)


| Keys     | Option |
| ----     | ------ |
| ↑        | turn up |
| ↓        | turn down |
| ←        | turn left |
| →        | turn right |
| space    | pause/resume |
| pageup   | speed up |
| pagedown | speed down |
